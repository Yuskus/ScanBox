# ScanBox

## Описание

Мини-сервис, позволяющий выполнять базовые функции по движению товаров.

#### Приход товара на склад

* Создание товара
* Генерация штрих-кода
* Печать штрих-кода
* Приход товара

#### Списание товара

* Создание отгрузки товара
* Сканирование товара
* Проверка на соответствие отсканированных товаров со списком товаров в отгрузке
* Списание товара

#### Отслеживание истории движения товара

* Ведение истории
* Просмотр истории
* Архив

## Логика

### Сущности

**1. Покупатели**

- ID (unique) (pk)
- ID контрагента (fk)

Ссылаются:
- никто

**2. Контрагент**

- ID (unique) (pk)
- ID тип контрагента
- Почтовый адрес
- Телефон
- E-mail (nullable)

Ссылаются:
- покупатель
- производитель
- поставщик
- документ
- юридическое
- физическое лицо

**3. Тип контрагента**

- ID (unique) (pk)
- Тип (юрлицо / физлицо)

Ссылаются:
- контрагент

**4. Документ (приход, уход)**

- ID (unique) (pk)
- Дата создания накладной
- ID Сотрудника склада (fk)
- ID типа накладной (fk)
- ID контрагента (fk)

Ссылаются:
- история движения

**5. Тип документа (накладной)**

- ID (unique) (pk)
- Название типа документа
- Описание состояния товара (на складе, реализован и тд)

Ссылаются:
- документ

**6. Физическое лицо**

- ID (unique) (pk)
- ID контрагента (fk)
- Фамилия
- Имя
- Отчество (nullable)

**7. Должность**  

- ID (unique) (pk)
- Название должности (unique)
- Описание обязанностей (nullable)
- Оклад

Ссылаются:
- сотрудники

**8. Юридическое лицо**

- ID (unique) (pk)
- ID контрагента (fk)
- Правовая форма (fk)
- Название юридического лица
- Фамилия руководителя
- Имя руководителя
- Отчество руководителя (nullable)
- ИНН
- КПП
- ОГРН
- Юридический адрес
- Контактное лицо (nullable)

**9. Правовые формы**

- ID (unique) (pk)
- Правовая форма
- Описание (nullable)

Ссылаются:
- юридическое лицо

**10. Производители**

- ID (unique) (pk)
- ID контрагента (fk)

**11. История движения**

- ID (unique) (pk)
- ID накладной (fk)
- ID товарной единицы (fk)

**12. Цены**

- ID товара (unique) (pk) (fk) 1к1
- Цена по себестоимости
- Цена розничная
- Цена оптовая

Ссылаются:
- тип продукта

**13. Категория товара**

- ID (unique) (pk)
- Категория (unique, index)
- Описание (nullable)

Ссылаются:
- тип товара

**14. Тип товара**

- ID (unique) (pk)
- Штрихкод (unique, index)
- Название продукта
- Длина
- Высота
- Ширина
- Вес
- Категория (fk)
- Производитель (fk)
- Цена товара (fk) 1к1

Ссылаются:
- товарная единица

**15. Товарная единица**

- ID (unique) (pk)
- Штрикод уникальный (unique, index)
- Дата производства
- Цена реализации
- ID типа товара (fk)
- ID поставщика (fk)

Ссылаются:
- история движения

**16. Поставщики**

- ID (unique) (pk)
- ID контрагента (fk)

**17. Работники склада**  

- ID (unique) (pk)
- ID должности (fk)
- Фамилия
- Имя
- Отчество (nullable)
- Дата рождения
- Дата найма
- Адрес
- Телефон

Ссылаются:
- документ
