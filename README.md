# ScanBox

## Описание

Мини-сервис, позволяющий выполнять базовые функции по движению товаров.

#### Приход товара на склад

* Создание товара
* Генерация штрих-кода
* Печать штрих-кода
* Приход товара

#### Списание товара

* Создание отгрузки товара
* Сканирование товара
* Проверка на соответствие отсканированных товаров со списком товаров в отгрузке
* Списание товара

#### Отслеживание истории движения товара

* Ведение истории
* Просмотр истории
* Архив

***

## Логика

### Сущности

**1. Товарная единица**

- UID (unique) (pk)
- ID типа товара
- ID поставщика
- Дата производства

**2. Тип товара**

- ID (unique) (pk)
- Штрихкод (unique)
- Название продукта
- Категория (fk)
- Поставщик (fk)
- Производитель (fk)
- Длина
- Высота
- Ширина
- Вес
- Количество 
- Цена товара (fk)

**3. Категория товара**

- ID (unique) (pk)
- Категория (unique)
- Описание (nullable)

**4. Поставщики**

- ID (unique) (pk)
- Юридическое лицо (fk)

**5. Правовые формы**

- ID (unique) (pk)
- Правовая форма
- Описание (nullable)

**6. Накладные прихода**

- ID (unique) (pk)
- Поставщик (fk)
- Сотрудник склада (fk)
- Дата поступления

**7. Накладные реализации**

- ID (unique) (pk)
- Покупатель (fk)
- Сотрудник склада (fk)
- Дата реализации

**8. Список товара на приход**  

- ID (unique) (pk)
- ID накладной прихода (fk)
- ID товара (fk)
- Количество

**9. Список товара на реализацию**  

- ID (unique) (pk)
- ID накладной реализации (fk)
- ID товара (fk)
- Количество

**10. Работники склада**  

- ID (unique) (pk)
- ID должности (fk)
- Фамилия
- Имя
- Отчество (nullable)
- Дата рождения
- Дата найма
- Адрес
- Телефон (nullable)

**11. Должности**  

- ID (unique) (pk)
- Название должности (unique)
- Описание обязанностей (nullable)
- Оклад

**12. Цены**

- ID товара (unique) (pk) (fk)
- Цена по себестоимости
- Цена розничная
- Цена оптовая

**13. Покупатель**

- ID (unique) (pk)
- Юридическое лицо (nullable) (fk)
- Фамилия
- Имя
- Отчество
- Почтовый адрес
- Телефон
- E-mail

**14. Юридическое лицо**

- ID (unique) (pk)
- Правовая форма (fk)
- Название юридического лица
- Фамилия руководителя
- Имя руководителя
- Отчество руководителя (nullable)
- ИНН
- КПП
- ОГРН
- Юридический адрес
- Почтовый адрес (nullable)
- Телефон
- E-mail
- Контактное лицо (nullable)

**15. Уценка**

- ID (unique) (pk)
- Штрихкод (unique)
- ID продукта (fk)
- Текущая цена
- Причина изменения цены (nullable)

**16. Производитель**

- ID (unique) (pk)
- ID юридического лица (fk)
